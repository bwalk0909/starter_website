Display and Box Sizing in CSS

Why the topic?

  • Display is the first thing to get right
  • You'll need to manage the relationship between display, float and position to really control layout and solve problems
  • Knowing your options with display can really help you be creative with CSS

Cut the fat, you need to know:

  1. block vs inline
  2. Margin/Padding on inline elements
  3. The four most common displays: inline, block, inline-block, none
  4. Using inline-block to make horizontal image lists
  5. Using box-sizing:border-box to change how margin/padding are calculated on block level elemnts
